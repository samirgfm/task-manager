Documentación Técnica: Aplicación Task-Manager

Requisitos Previos

Para configurar y ejecutar la aplicación Task-Manager, asegúrate de tener instalados los siguientes elementos en tu máquina:

  • Docker (v26.0)
  • Docker Compose (v2.1)

Pasos para Levantar la Aplicación

  1. Cargar las Imágenes Docker:

    2. Antes de levantar la aplicación, carga las imágenes Docker utilizando el siguiente comando:

    docker load -i task-manager.tar
  2. Ejecutar Docker Compose:

    3. En el directorio donde se encuentra el archivo docker-compose.yml, ejecuta el siguiente comando:

    docker-compose up -d

    Este comando levantará la aplicación en modo daemon, lo que significa que seguirá ejecutándose en segundo plano.

Acceso a la API

Una vez que la aplicación esté levantada, podrás acceder a la API. Para obtener más información sobre cómo interactuar con la API, consulta la documentación Swagger en el archivo task-manager-api.yml.

Ejemplos de Uso de la API con cURL

A continuación, se presentan algunos ejemplos de cómo utilizar la API utilizando el comando cURL:

  1. Obtener Todas las Tareas (getAll):
    2. curl 'http://localhost:8080/tasks'
  2. Obtener una Tarea por su ID (getById):
    3. curl 'http://localhost:8080/tasks/{id}'
  3. Crear una Nueva Tarea (Create task):
    4. curl -X POST http://localhost:8080/tasks \
     -H "Content-Type: application/json" \
     -d '{
           "title": "Nueva Tarea 1",
           "description": "Descripción de la nueva tarea",
           "dueDate": "2024-06-30",
           "tags": ["etiqueta1", "etiqueta2"]
         }'
  4. Actualizar una Tarea (updateTask):
    5. curl -X PUT http://localhost:8080/tasks/6664448e558e274bc38d4688 \
     -H "Content-Type: application/json" \
     -d '{
           "title": "Tarea Actualizada",
           "description": "Descripción actualizada",
           "dueDate": "2024-06-30",
           "tags": ["etiqueta1", "etiqueta2"],
           "status": "COMPLETED"
         }'
  5. Eliminar una Tarea por su ID (deleteById):
    6. curl -X DELETE http://localhost:8080/tasks/{id}

Recuerda reemplazar {id} con el ID correspondiente de la tarea que deseas obtener, actualizar o eliminar.
